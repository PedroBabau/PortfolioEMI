# 🌐 PORTFOLIO EMI – Projetos de Edições de Multimédia Interativas

Este repositório contém um portfólio interativo desenvolvido no âmbito da Unidade Curricular **Edições de Multimédia Interativas**. O portfólio apresenta **8 projetos individuais** organizados por áreas temáticas.

---

## 🎯 Estrutura do Portfólio

| # | Projeto | Área | Tecnologias |
|---|---------|------|-------------|
| 1 | Particle Storm | Coding Interativo | JavaScript, Canvas, WebGL |
| 2 | Sound Painter | Coding Interativo | Web Audio API, Canvas, JavaScript |
| 3 | Orbital Viewer 3D | Experiências 3D | Unity, WebGL, C# |
| 4 | Camera Director | Experiências 3D | Unity, WebGL, Cinemachine |
| 5 | GeoPulse | Dashboards | p5.js, JavaScript, API |
| 6 | API Explorer | Dashboards | p5.js, JavaScript, JSONPlaceholder |
| 7 | AR Animal Care | AR & VR | A-Frame, AR.js, JavaScript |
| 8 | VR Gundam Hangar | AR & VR | A-Frame, Three.js, WebXR |

## 📁 Projetos por Área

### ⚡ Coding Interativo

#### 1. Particle Storm
| Campo | Descrição |
|-------|-----------|
| **Objetivo** | Simular um sistema de partículas com comportamento emergente e interação em tempo real. |
| **Tecnologias** | JavaScript (Canvas/WebGL), HTML5, CSS3 |
| **Funcionalidades** | - Geração de centenas de partículas com física básica <br>- Interação com rato/teclado <br>- Efeitos visuais neon e rastros luminosos |
| **Acesso** | [Particle Storm Demo](https://pedrobabau.github.io/Coding_Interativo/projeto1/index.html) |
| **Repositório** | [Coding_Interativo](https://github.com/PedroBabau/Coding_Interativo) |

#### 2. Sound Painter
| Campo | Descrição |
|-------|-----------|
| **Objetivo** | Pintar visualmente utilizando input de áudio (microfone ou ficheiros). |
| **Tecnologias** | Web Audio API, Canvas, JavaScript |
| **Funcionalidades** | - Análise de frequências em tempo real <br>- Mapeamento som → cor/intensidade <br>- Exportação de frames como imagem |
| **Acesso** | [Sound Painter Demo](https://pedrobabau.github.io/Coding_Interativo/projeto2/index.html) |
| **Repositório** | [Coding_Interativo](https://github.com/PedroBabau/Coding_Interativo) |

## 🎨 Interface Gundam 00 – Funcionalidades Técnicas

| Elemento | Descrição |
|----------|-----------|
| **Cores** | Azul GN (#4a9eff), Laranja (#ffb233), Branco metálico, Azul escuro |
| **Efeito GN Particles** | Partículas de fundo com animação suave |
| **Grid Cockpit** | Linhas de fundo estilo painel de controlo |
| **Header Animado** | Gradiente com animação de deslocamento |
| **Cards** | Efeito glassmorphism com bordas azuis/laranja |
| **Botões** | Estilo transparente com hover em azul GN |
| **Grid Responsivo** | 4 colunas (desktop) → 2 colunas (tablet) → 1 coluna (mobile) |
| **Stats Bar** | Contador de projetos, tecnologias, repositórios e áreas |
